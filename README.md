# ATM Simulation

A simple menu-driven ATM Simulation project written in Python.

## Features

- Display current balance
- Withdraw money
- Deposit money
- View transaction statement
- Infinite menu loop using `while True`
- Simple and beginner-friendly code
- No OOP used

## Requirements

- Python 3.x

## Description

This ATM Simulation allows users to check their balance, deposit money, withdraw money, and view their transaction history. The program runs continuously until the user chooses the exit option.
